# Regression Suite

## Homepage

- Homepage loads successfully.
- Header is visible.
- Search box is visible.
- Main content is visible.
- Footer is visible.
- Cookie banner is handled safely if present.

## Search

- Safe search term opens results.
- Alternative search term opens results.
- Search does not require login.
- Results page shows listing content.

## Listing

- Product cards are visible.
- Filter or sort controls are visible when available.
- First product can be opened safely.

## Product Detail

- Product detail page loads.
- Product title is visible.
- Price or product information is visible.
- Add-to-cart/favorite CTA is observed only.
- Purchase flow is not started.

## Login Boundary

- Login page can be opened if public route is available.
- Login form or login boundary is visible.
- No real credentials are used.
- No form is submitted.

## Cart Boundary

- Cart route or cart boundary is observed.
- Checkout boundary is documented.
- No checkout, payment, or final submission is performed.

## Footer/Help

- Footer is visible.
- Safe help/legal link opens public informational content.

## Mobile

- Homepage loads on mobile viewport.
- Search is reachable on mobile.
- Listing can be reached safely when public behavior allows it.
