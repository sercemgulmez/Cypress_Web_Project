## How to Apply via GitHub Web UI

1. Open the repository main page on GitHub.
2. In the **About** section on the right side, click the **gear icon** (⚙️).
3. In the **Description** field, paste the recommended description above.
4. In the **Topics** field, add each topic from the list above one by one.
5. Click **Save changes**.

---

## Recommended Social Preview (optional)

If you want to add a social preview image (shown when the link is shared on LinkedIn or Twitter):

1. Go to **Settings** → **Options** → **Social Preview**.
2. Upload a clean, professional image (1280×640 px recommended).
3. Content suggestion: project name, key technologies (Cypress, TypeScript), and a one-line description.
